Project Goals

0) a top devgrammer portfolio in 100% plain old html, css, and javascript.

1) a stream entropy as an interesting water fountain except in pixels; it includes "water cooler" discussion topics to further conversations about subvind.

2) a backend server coded in nest.js that manages ERP database records in PostgreSQL, event-driven microservices with RabbitMQ, and integrates well with 3rd party APIs like stripe for payments and twilio for sms.

3) a frontend client which is an ERP admin area for employees; because the backend is headless all customer facing content is done on it's own CMS.

4) a main store that showcases products.

5) a sync inamlang lexer, parser, and interpreter that converts a strings of text (blocks of custom code formatted in xml, cli, ts, or json) and compiles them into a running script.

6) a base autoide for visualizing business logic in a diagram format; easily picture how ERP events are triggered, such as customer created or order recieved, and then handled throughout the system.

7) a cli inam for importing/exporting data in and out of user/organizations and managing license keys.

8) a tv flowtube with playlists that cover everything subvind.

Target Audience

Someone that is interested in flow-based programming with a developer mindset. They think in terms of nodes, anchors, and edges. They have an itching desire to make impovements. They are interested in seeing their code visually through diagram technology when it comes to building a program; yet, terminal user interfaces are their friend.

How to use

Verify node.js package manager: 
$ npm --version
Then install this terminal user interface: 
$ npm install --global inam
Read the manual for documentation: 
$ inam --help
